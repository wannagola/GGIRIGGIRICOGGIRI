# 동영상 추가 방법

이 폴더에 동영상 파일을 넣으면 끼리끼리 페이지의 "🎥 동영상" 탭에서 자동으로 표시됩니다.

## 지원하는 동영상 형식
- MP4 (.mp4)
- WebM (.webm)
- MOV (.mov)
- AVI (.avi)

## 사용 방법
1. 이 폴더(`public/assets/videos/`)에 동영상 파일을 복사합니다
2. 페이지를 새로고침하면 자동으로 동영상이 나타납니다
3. 동영상 썸네일을 클릭하면 재생됩니다

## 예시
만약 `my-video.mp4` 파일을 이 폴더에 넣으면:
- 자동으로 동영상 목록에 추가됩니다
- 썸네일과 함께 그리드로 표시됩니다
- 클릭하면 모달에서 재생됩니다
