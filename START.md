# 🚀 빠른 시작 가이드

## 지금 바로 실행하기!

터미널을 열고 아래 명령어를 **순서대로** 실행하세요:

### Step 1: 프로젝트 폴더로 이동
```bash
cd "/Users/wanipark1004/Desktop/끼리끼리코끼리"
```

### Step 2: 패키지 설치 (이미 했다면 건너뛰기)
```bash
npm install
```

### Step 3: 개발 서버 시작 ⭐️
```bash
npm run dev
```

### Step 4: 브라우저 열기
브라우저에서 접속: **http://localhost:3000**


### 브라우저에서 뭐가 보이나요?

#### ✅ 정상:
- 하늘과 대지 그라데이션 배경
- 중앙에 귀여운 코끼리 캐릭터

#### ❌ 문제:

- **버튼이 없음**: F12 눌러서 콘솔 에러 확인
- **페이지가 안 뜸**: 포트 번호 확인 (3000)

## 🎤 마이크 권한 허용하기

녹음 버튼을 처음 누르면 브라우저에서 이런 팝업이 나타납니다:

```
localhost에서 마이크를 사용하려고 합니다.
[차단] [허용]
```

👉 **반드시 [허용]을 클릭하세요!**

차단했다면?
1. 주소창 옆 자물쇠 🔒 아이콘 클릭
2. 마이크 > 허용으로 변경
3. 페이지 새로고침 (Cmd+R 또는 F5)

---

## 🎯 사용 방법

1. **빨간 버튼** 클릭 → 녹음 시작
2. 말하기 → 다시 **빨간 버튼** 클릭 → 녹음 종료
3. 자동으로 코끼리 목소리로 변환 (2-3초 소요)
4. **초록 버튼** 클릭 → 변조된 목소리 듣기
5. 코끼리 클릭 → "이잉~" 소리와 움찔 애니메이션

## ❓ 여전히 안 되나요?

### 1. 터미널 확인
`npm run dev` 명령어를 실행했나요?
- ✅ 실행함 → 터미널에 에러 메시지가 있나요?
- ❌ 안 함 → **지금 바로 실행하세요!**

### 2. 브라우저 콘솔 확인
1. 브라우저에서 F12 (Mac: Cmd+Option+I)
2. Console 탭 클릭
3. 빨간 에러 메시지가 있나요?
   - 있으면 → 에러 메시지 복사해서 알려주세요
   - 없으면 → "Current status: idle" 메시지가 보여야 정상

### 3. 캐시 삭제 후 새로고침
- Mac: Cmd+Shift+R
- Windows: Ctrl+Shift+R

### 4. 그래도 안 되면?
```bash
# 완전히 다시 시작
rm -rf node_modules package-lock.json .next
npm install
npm run dev
```
